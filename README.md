# Dim for X

> Brings back the classic Dim theme to X (Twitter) ‚Äî with full color customization.

X removed the Dim theme in February 2026, leaving only pitch black and white. **Dim for X** restores it, with 7 presets and a full custom color picker.

üåê **[dimforx.com](https://dimforx.com)**

---

## Features

- One-click toggle
- 7 built-in presets: Dim, Midnight, Nordic, Forest, Dracula, Coffee, Space
- Custom color via HSV canvas picker, hue slider, and hex input
- Covers the full X interface ‚Äî feed, DMs, modals, GIF picker, search, X Pro
- DOM scanner catches dynamically loaded inline black backgrounds
- Syncs with X's dark mode ‚Äî activates on `body.LightsOut`, suspends on light mode
- Flash-free load via `preload.css`
- Settings sync across devices (Chrome Sync)
- Manifest V3
